# Renewable-energy-analysis
Explored renewable energy sales data in Excel to uncover trends, compare regional performance, and summarize key insights using Pivot Tables.

Through this project I have explored renewable energy sales data using Microsoft Excel. My goal was to analyze sales performance across different regions and energy categories, and identify trends in the data, as well as create summary reports that make the information easier to understand and compare.

Using Excel I organized, filtered, and analyzed the data. I created Pivot Tables to summarize sales performance, compare results across regions and categories, and highlight key trends within the dataset. Furthermore, I gained experience using Pivot Tables, data filtering, data aggregation, and reporting techniques to transform raw data into meaningful business insights.

The analysis revealed differences in sales performance across regions and energy categories. By summarizing the data through Pivot Tables, I was able to identify trends, compare performance, and better understand which areas contributed most to overall sales. The project basically demonstrates how Excel can be used to support data-driven decision-making and business reporting.
